About Bee Planet Connection

We serve global apiculture as an open hub for knowledge, tools, a directory and discussion — championing environmental stewardship.

What BPC is

  • A free, open knowledge base for apiculture.
  • A practical toolkit for beekeepers.
  • A global directory of organisations.
  • A forum for thoughtful discussion.

What BPC isn’t

  • Not a trade marketplace.
  • Not a replacement for local regulation or veterinary advice.
  • Not a closed community — we welcome constructive contributions.

Contribute

  • Suggest edits via GitHub pull requests.
  • Propose directory additions.
  • Share tools and templates.
  • Join discussions and mentor newcomers.
